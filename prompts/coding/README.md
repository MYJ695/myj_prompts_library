# 编程相关 Prompts

本目录收录编程开发相关的提示词模板。

## 📋 提示词列表

- `fpga-verilog-expert.md` - 务实派 FPGA 开发与架构专家（Verilog-2001）
- `code-review.md` - 代码审查提示词
- `code-optimization.md` - 代码优化提示词
- `bug-debugging.md` - Bug 调试提示词
- `tech-doc-writing.md` - 技术文档编写提示词

## 💡 使用建议

根据具体的编程场景选择合适的提示词，并根据实际代码和需求进行调整。对于 FPGA 和硬件设计，推荐使用 FPGA Verilog 专家提示词。
