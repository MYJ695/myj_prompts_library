# MYJ Prompts Library

[![GitHub](https://img.shields.io/badge/GitHub-MYJ695-blue?logo=github)](https://github.com/MYJ695/myj_prompts_library)

一个精心整理的 AI Prompts 提示词库，收录了我在日常工作和学习中常用的高质量提示词模板。

## 📖 简介

本仓库旨在收集和分享各类场景下的优质 AI 提示词，帮助提升与 AI 的交互效率。所有提示词均经过实际使用验证，持续更新优化。

## 📂 目录结构

```
myj_prompts_library/
├── README.md                 # 项目说明文档
├── prompts/                  # 提示词目录
│   ├── coding/              # 编程相关
│   ├── writing/             # 写作相关
│   ├── analysis/            # 分析相关
│   ├── translation/         # 翻译相关
│   └── creative/            # 创意相关
└── templates/               # 提示词模板
```

## 🚀 快速开始

1. 浏览 `prompts/` 目录，找到你需要的场景分类
2. 选择合适的提示词模板
3. 根据实际需求调整参数和内容
4. 在你的 AI 工具中使用
